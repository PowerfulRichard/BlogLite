n%m

题目描述

给定两个整数n，m, 1≤n≤220, 1≤m≤2014，求

∑i=1n(i%m)=1%m+2%m+3%m...+n%m

的值。

输入

第一行有一个整数N表示有N组测试数据。 每组测试数据都有两个整数n，m, 1≤n≤220, 1≤m≤2014

输出

求出每组

∑i=1n(i%m)=1%m+2%m+3%m...+n%m

的值。 

#include <iostream>
using namespace std;
int main() {
    long long c, b, a, n, m;
    cin >> a;
    while (a--) {
        cin >> n >> m;
        b = n % m, c = n / m;
        cout << m * (m - 1) / 2 * c + b * (b + 1) / 2 << "\n";
    }
}
