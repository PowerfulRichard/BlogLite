整数模

题目描述

a除以m的余数称为a对于m的模，一般记做 amodb。求apmodm。

输入

输入数据中含有一些数据组，每个数据组占一行，包括a、p、m（1≤a1≤a,p≤231−1,1≤m≤216）三个整数，若三个数都为，则表示输入结束。

输出

针对每组数据，输出apmodm。，每个结果占一行。 
#include<bits/stdc++.h>
using namespace std;
typedef long long ll;
int main() {
    ll a, b, c;
    while (scanf("%lld%lld%lld", &a, &b, &c)) {
        ll d = 1;
        if (a == 0 && b == 0 && c == 0)break;
        for (int i = 0; i < b; i++)d = d * a % c;
        printf("%lld\n", d);
    }
}
