敲7

题目描述

输出7和7的倍数，还有包含7的数字例如17，27，37,...,70，71，72，73…

输入

一个整数NN。N≤3000

输出

从小到大排列的不大于N的与7有关的数字，每行一个。 
#include<bits/stdc++.h>
using namespace std;
int main(){
    int n,i=1;
    cin>>n;
    while(i<=n){
        if(i/10==7||i%7==0||i%10==7||i/100==7||i%1000==7){
            cout<<i<<endl;
        }
        i++;
    }
    
    return 0;
}
