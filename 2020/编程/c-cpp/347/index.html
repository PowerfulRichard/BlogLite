最近平方数

题目描述

如果一个整数可以写成另一个整数的平方，则说它是一个完全平方数。比如1,4,9,16是完全平方数。
输入一个整数，找到一个离它最近的完全平方数。

输入

仅有一组测式数据，输入一个整数NN,  ≤N≤100000。

输出

输出离它最近的完全平方数,如果N就是完全平方数，则输出N。 
#include<bits/stdc++.h>
using namespace std;
int main(){
    int n,a=1,b,c,nn;
    cin>>n;
    b=n;
    c=n;
    while(a*a!=b){
          while(a*a<b){
            a++;
          }
          if(a*a!=b){
                a=1;
            b--;
          } 
    }
    while(a*a!=c){
          while(a*a<c){
            a++;
          }
          if(a*a!=c){
            a=1;
            c++;
          } 
    }
    if((c-n)>(n-b)){cout<<b;}
    else{cout<<c;}
    return 0;
}
